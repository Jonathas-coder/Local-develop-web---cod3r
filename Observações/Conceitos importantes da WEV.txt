HTTP - PROTOCOLO DE TRANSFERENCIA DE HIPER TEXTOS.

SERVER SIDE - Aplicação que do lado backend que gera dados a partir de uma linguagem;
CLIENT SERVER - Aplicação front end que consome dados e exibe os dados HTML gerados pelo server side;

CONTEUDO ESTÁTICO - Arquivos do server side é servido ao client side e não pode ser alterados pelo client, apenas no código fonte;
CONTEUDO DINAMICO - Alterações que podem ser feitas a partir dos arquivos server side, NO CLIENT SIDE...
Por exemplo: No server eu gero arquivos HTML com python, e o client pode alterar esse arquivo com JS no client, sem necessariamente tocar
no codigo fonte.
